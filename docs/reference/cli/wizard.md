# Wizard CLI Reference

A wizard is an agentic helper that can assist users in performing tasks. 
The `wizard` CLI allows you to run wizards and to interact with them.
For managing wizards, see the `wizards` CLI reference.

## CLI Usage

```bash
geenii wizard info <wizard-name> [options]
geenii wizard run <wizard-name> [options]
geenii wizard history <wizard-name> [options]
geenii wizard message <wizard-name> [options]
geenii wizard chat <wizard-name> [options]
```
