# Skills CLI Reference

Skills are reusable tools that can be used by wizards to perform specific tasks.
The `skills` CLI allows you to manage skills, including creating, updating, and deleting them

## CLI Usage

```bash
geenii skills list [options]
geenii skills create <skill-name> [options]
geenii skills update <skill-name> [options]
geenii skills delete <skill-name> [options]
```
